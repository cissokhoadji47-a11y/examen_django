Centre d'aide

Pour créer un projet :

  1. Connectez-vous à votre compte
  2. Cliquez sur "Mes Projets" dans le menu
  3. Cliquez sur le bouton "Nouveau Projet"
  4. Remplissez le titre et la description
  5. Cliquez sur "Créer le projet"

Pour ajouter une tâche à un projet :

  1. Ouvrez le projet concerné
  2. Cliquez sur "Nouvelle tâche"
  3. Remplissez le titre, la description, la date limite
  4. Assignez la tâche à un membre (optionnel)
  5. Cliquez sur "Créer la tâche"

Les professeurs peuvent bénéficier de primes selon leurs performances :

  • 30 000 FCFA : 90% des tâches terminées dans les délais
  • 100 000 FCFA : 100% des tâches terminées dans les délais

Les statistiques sont calculées automatiquement et visibles dans la section "Mes statistiques".
